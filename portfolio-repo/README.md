# Kedar Mahajan — Portfolio

Cyberpunk-aesthetic portfolio website for a Prompt Engineer & AI Visual Designer.

## Structure

```
kedar-mahajan-portfolio/
├── index.html              ← Main entry point
├── css/
│   └── style.css           ← All styles
├── js/
│   └── main.js             ← Navigation, lightbox logic
└── assets/
    └── images/
        ├── frontline/      ← Frontline Force character art
        ├── steampunk/      ← Steampunk Electricity Plant
        ├── post_apoc/      ← Post-Apocalyptic Game Designs
        ├── minneapolis/    ← Minneapolis Skyline
        └── strategy/       ← M.A.V. Strategy Card Game
```

## How to Run

Just open `index.html` in any browser — no build step, no dependencies.

> **Note:** Because images are loaded via relative paths, open the file through a
> local server or directly from the folder (not a standalone file drag).  
> Quick option: `npx serve .` or use VS Code Live Server.

## Deploy

Upload the entire folder to any static host:
- **GitHub Pages** — push to a repo, enable Pages from Settings
- **Netlify** — drag the folder into netlify.com/drop
- **Vercel** — `vercel .` from the folder
